# FitLife - Fitness Website (Assignment 2.1)

A modern, responsive fitness website created with pure HTML and CSS.

## Overview

FitLife is a professional fitness gym website showcasing services, membership programs, testimonials, and a contact form. This project demonstrates modern web design principles using only HTML and CSS without any frameworks.

## Features

- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Navigation Bar** - Sticky header with smooth navigation links
- **Hero Section** - Eye-catching banner with call-to-action button
- **Services Section** - Showcasing 4 main fitness services with icons
- **Programs Section** - Three membership tiers with pricing and features
- **Testimonials Section** - Client reviews and ratings
- **Contact Form** - Simple form for customer inquiries
- **Modern Styling** - Gradient backgrounds, hover effects, and smooth transitions

## Responsive Breakpoints

- **Desktop**: Full layout
- **Tablet** (768px and below): Slight adjustments
- **Mobile** (480px and below): Optimized for small screens
